## Install react bootstrap
1.สร้างโปรเจค
npx create-react-app ชื่อโปรเจค

2.ติดตั้ง bootstrap =>
npm install react-bootstrap bootstrap

3.เอาคำด้านล่างไว้ที่ src/index.js or App.js file =>
import 'bootstrap/dist/css/bootstrap.min.css';
